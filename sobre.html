Sobre o Projeto

Este projeto foi desenvolvido durante o curso de mestrado em Biblioteconomia pela discente Luciana Esteves.

Possui o objetivo de criar um sistema eficiente para consulta de perfis dos profissionais. Através do Supabase, armazenamos informações de pessoas e disponibilizamos uma interface intuitiva para pesquisa e consulta.

O sistema permite que os usuários busquem por nomes e acessem detalhes sobre cada perfil armazenado no banco de dados.
